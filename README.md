# My Personal Planner • Dec 2025 – Jan 2027

A clean, beautiful, no-time-block digital planner that saves everything in your browser.

Live version → https://yourusername.github.io/my-2025-2027-planner/

![screenshot](assets/screenshot.png)  ← (add one later)

### Features
- Full monthly calendars with event dots
- 8-slot weekly view (7 days + Weekly Goals)
- Completely free-form — no time blocking
- Auto-saves everything forever using your browser storage
- Works offline
- Gorgeous typography & colors
- Extra sections: Big Goals, Habits, Gratitude

Just open the link and start typing. All your data stays 100% private in your browser.

Made with ❤ using plain HTML + CSS + JavaScript (luxon.js from CDN)
